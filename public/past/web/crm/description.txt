Cross-Platform CRM System for Financial Management
Web & Mobile Application | Admin & Client Portal

Developed a comprehensive CRM platform tailored for financial operations, supporting both administrative and client-facing workflows. The system integrates modules for contract and contact management, task tracking, invoice processing, and role-based permissions.

Key Features:
- Multi-role access control system with tailored permission levels
- Contract lifecycle support including NDA signing and digital document handling
- Real-time features: integrated chat (WebRTC), live push notifications, and location tracking
- Advanced time tracking with geo-based session logs
- Multiple customizable dashboard views for different user types (admin/client)

The CRM was deployed as both a web and mobile solution, ensuring seamless user experience across devices. It was built to scale, with a modular architecture that allows for quick feature expansion and maintenance.

Used: React, Next.js, MUI, Bootstrap, Redux, WebRTC for frontend, Django, PostgreSQL for backend, React Native for Mobile, hosted on AWS(EC2, S3, Lambda, RDS)
