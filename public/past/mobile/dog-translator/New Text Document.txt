An interactive mobile application that enables simulated communication between users and their dogs through audio recording, sound transformation, and API-driven interpretation services.

Role: React Native / iOS Developer

Features
- Implemented real-time voice recording and playback using React Native audio libraries with native iOS AVFoundation integration for smooth sound capture and output.
- Integrated bark-to-text and voice transformation APIs using RESTful services with Axios, handling audio file uploads, response parsing, and structured UI rendering.
- Built dog whistle functionality using native audio frequency generation modules, allowing dynamic tone adjustment within safe frequency ranges.
- Developed secure user authentication and session management using Firebase Authentication, enabling personalized user experiences.
- Implemented cloud data storage for recording history and user activity using Firebase Firestore.
- Integrated Push Notifications via Firebase Cloud Messaging to improve user engagement and feature updates.
- Optimized cross-device performance and ensured consistent behavior across different iOS versions using TypeScript-based state management and async handling patterns.

Challenge:
Handling real-time audio recording and playback while maintaining smooth UI performance across different iPhone models required careful async processing and memory management. Managing large audio file uploads to external APIs without blocking the main thread was another key challenge, which was solved through optimized file handling and background request execution. Ensuring consistent audio behavior across iOS versions required additional native-level adjustments and device testing.

Tech Stack: React Native, TypeScript, Firebase (Authentication, Firestore, Cloud Messaging), REST APIs (Axios), AVFoundation (iOS Native Modules) ,Audio Processing Libraries
